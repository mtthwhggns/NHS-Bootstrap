Bootstrap for NHS Organisations

Quickly create simple websites and apps without a designer, using the web's most popular framework, ready-styled to match the NHS Brand Guidelines

Download GitHub Project

The Styles

All of the syntax needed matches exactly that of Bootstrap, so please refer to the Bootstrap getting started guide and documentation for details of the code you need to incorporate various styles and components, just replace the bootstrap files with the NHS Bootstrap release.

Need Help?

If you need help with the Bootstrap syntax, a good place to start is StackOverflow, use the twitter-bootstrap tag on your questions. If you have a problem related to the NHS-specific styles, feel free to open an issue on Github.
