Signage Mockupsbeta

We've included some classes which allow you to create signage mockups in any NHS colour.

You just need to start with a div with the class of sign and either one-line-horizontal or three-line-rectangle. You can optionally add the class destination-centre to centre the sign text, and destination-right to right-align it. There's no class needed to left align the text.

Inside this, just include the text of the sign, inside a span with the class destination.

You can add an arrow to the sign, using glyphicons. If you want it to appear on the left, just before the text, include the glyphicon span code inside the destination span. If you want to display it on the right, add the class of arrow-right to the destination span and the class glyphicon-arrow-right to the glyphicon span.
