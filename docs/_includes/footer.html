The Project

The downloadable releases are designed to be dropped in in place of Twitter Bootstrap, so there are no extra files and no complexities over which files to replace. That doesn't mean we want to replace Bootstrap! When we generate a copy of the files, we use the most recent release of Bootstrap, and some files are not modified at all.

The Creator

The project was created by Matthew Higgins, who noticed while creating another project (soon to be open source) that a big part of his time had been spent creating a Bootstrap theme, which could be useful in lots of projects, and decided to spin this out.
