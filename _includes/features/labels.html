Labels

Labels are a core part of Bootstrap, we've just added some of our own colours.

We've created a set of labels featuring all of the NHS Brand colours. Just add the appropriate class, from the list on the left, to a span which also has the label class

This does mean that there are some duplicates, for example, .label-primary is the same as .label-nhs-blue
